# Management Tools

Free project management, productivity, and analytics tools for students.

| Tool/Service | Description | Access Link |
|--------------|-------------|-------------|
| **Atlassian Jira** | Project management and issue tracking software | [Atlassian for Students](https://www.atlassian.com/software/free) |
| **Notion Student Pack** | All-in-one workspace for notes, tasks, and collaboration | [Notion Education](https://www.notion.so/product/notion-for-education) |
| **Gurobi** | Mathematical optimization solver for complex problems | [Gurobi Academic License](https://www.gurobi.com/academia/academic-program-and-licenses/) |
| **Tableau Desktop** | Data visualization and business intelligence (1 year free) | [Tableau for Students](https://www.tableau.com/academic/students) |

## Tips for Getting Started

- **Jira**: Perfect for managing group projects and tracking assignments
- **Notion**: Great for organizing notes, creating wikis, and collaborative documentation
- **Gurobi**: Essential for operations research and optimization courses
- **Tableau**: Ideal for data analysis projects and creating interactive dashboards
