# Development Tools

Free development tools and platforms available to students.

| Tool/Service | Description | Access Link |
|--------------|-------------|-------------|
| **GitHub Student Developer Pack** | Comprehensive bundle of developer tools and services | [GitHub Education](https://education.github.com/pack) |
| **Azure Cloud Services** | Microsoft's cloud computing platform with student credits | [Azure for Students](https://azure.microsoft.com/en-us/free/students/) |
| **Oracle Cloud** | Cloud infrastructure and platform services | [Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/) |
| **AWS Account** | Amazon Web Services with educational credits | [AWS Academy](https://go.aws/4piBf1h) |
| **Google Firebase** | Backend-as-a-Service platform for web and mobile apps | [Firebase](https://firebase.google.com/) |
| **Google Gemini API** | Google's AI model API for developers | [Google AI Studio](https://ai.google.dev/) |
| **Google Colab** | Free Jupyter notebook environment with GPU access | [Google Colab](https://colab.research.google.com/) |
| **Databricks** | ETL Pipelines and data analytics platform | [Databricks Community Edition](https://community.cloud.databricks.com/) |
| **Perplexity Pro** | AI-powered search and research tool (2 months free) | [Referral Link](https://plex.it/referrals/NR7SLEM7) |
| **Autodesk** | CAD, 3D modeling, and design software | [Autodesk Education](https://www.autodesk.com/education/home) |
| **JetBrains** | Professional IDEs for various programming languages | [JetBrains Student License](https://www.jetbrains.com/community/education/) |
| **Antigravity IDE** | Google's modern development environment | [Antigravity](https://labs.google/antigravity) |
